# Quirgo CLI

Quirgo is a tool built to easily configure repositories' secrets and variables.

## Installation

```
yarn add -g quirgo-cli
```

or

```
pnpm add -g quirgo-cli
```

or

```
npm install -g quirgo-cli
```

## Usage

To start using Quirgo, run:

```
quirgo-cli
```

### Commands

- vars
- secrets
